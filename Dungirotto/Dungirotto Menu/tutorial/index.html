Bem-vindo ao Dungeon Crawler!

Este é um jogo de aventura em que você explora diferentes masmorras.

Controles

  • Pressione a tecla "W" para andar para cima.
  • Pressione a tecla "A" para andar para esquerda.
  • Pressione a tecla "S" para andar para baixo.
  • Pressione a tecla "D" para andar para direita.
  • Pressione a tecla "I" para interagir com objetos (O,0,U,|,$,<,>,#).

Objetivo

Seu objetivo é capturar uma chave em meio de vários desafios para assim abrir uma porta trancada.

Dicas

  • Não toque nos espinhos (#), eles fazem com que você se machuque.
  • As chaves (@) estão escondidas em salas secretas, então explore bem cada sala.
